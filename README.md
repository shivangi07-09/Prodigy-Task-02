# REST API Development Project

## Description
This project is a REST API developed using Node.js, Express.js, MongoDB, and Mongoose.

## Features
- Create User (POST)
- Get All Users (GET)
- Get User by ID (GET)
- Update User (PUT)
- Delete User (DELETE)

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
- Thunder Client

## API Endpoints

POST /api/users

GET /api/users

GET /api/users/:id

PUT /api/users/:id

DELETE /api/users/:id
